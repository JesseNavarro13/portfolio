# Portfolio Website

This is my personal portfolio project showcasing my projects and skills. Built with React for the frontend and Spring Boot for the backend.

---

## Features
- Home Page - Displays personal info and portfolio overview.
- Projects Page - Shows a list of projects with cards.
- Admin Dashboard - Login-protected area to manage projects.
- Login Authentication - Admin users can log in and manage content.
- Responsive Design - Works on desktop and mobile devices.

---

## Tech Stack
- Frontend: React, React Router DOM, Vite
- Backend: Spring Boot (JWT authentication)
- Styling: CSS / Tailwind CSS
- Data Storage: PostgreSQL
